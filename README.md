# StyleHub — Your AI Fashion Stylist 👗✨

> Built for **Aavishkar 2026 Open Hackathon** | 22nd April 2026

## 🚀 What is StyleHub?

AI-powered fashion styling web app — color combinations, outfit advice, weather-based dressing, and more.

## ✨ Features

| Feature | Description |
|---|---|
| 🎬 Splash Screen | Animated golden bubble-letter intro |
| 🔐 Auth System | Login & Sign Up with localStorage |
| 👗 Outfit Builder | Mix & score outfit combinations |
| 🎨 Color Picker | Pick 2 colors, get compatibility score |
| 🌤️ Weather Outfit | Real weather → outfit suggestion |
| 🧠 AI Style Advisor | Chat-based fashion advisor |
| ❤️ Wishlist | Save your favourite outfits |
| 👤 Profile | Stats, dark mode & all features |

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- localStorage — Auth & data persistence
- Open-Meteo API — Real weather data (free)

## ⚡ Run

Open `index.html` in any browser — no installation needed.
